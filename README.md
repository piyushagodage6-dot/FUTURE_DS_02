### Objective
To analyze customer churn and retention patterns and identify key drivers that impact customer lifetime and business revenue.

### Dataset
Customer churn dataset containing customer demographics, subscription details, tenure, payment methods, and churn status.

### Tools Used
- Power BI
- Microsoft Excel

### Key KPIs
- Total Customers
- Churned Customers
- Churn Rate (%)
- Average Tenure
- Monthly Charges

### Analysis Performed
- Churn analysis by contract type
- Churn analysis by payment method
- Customer tenure trends
- Customer lifetime patterns

### Key Insights
- Month-to-month contract customers have the highest churn.
- Customers using electronic check show higher churn.
- Longer-tenure customers are more likely to stay.
- Higher monthly charges increase churn risk.

### Business Recommendations
- Encourage long-term contracts with incentives.
- Promote auto-payment methods.
- Improve early-stage customer engagement.

### Outcome
This project demonstrates real-world customer retention and churn analysis used in subscription-based businesses.
